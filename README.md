# Northwind Traders Power BI Dashboard

## Overview

This project is a multi-page interactive dashboard built using **Microsoft Power BI** with the **Northwind Traders Dataset**.

The dashboard helps analyze:

* Sales performance
* Product insights
* Shipment analysis
* Employee performance
* Operational efficiency

---

# Dashboard Pages

## Page 1 — Executive Overview

This page shows the overall business performance.

### Includes:

* Total Sales
* Total Orders
* Average Order Value
* Monthly Sales Trend
* Category Performance
* Shipment Status

---

## Page 2 — Product & Category Analysis

This page focuses on product-level insights.

### Includes:

* Top Selling Products
* Category Contribution
* Product Sales Analysis
* Quantity vs Sales Analysis
* Discount Impact

---

## Page 3 — Operations Dashboard

This page focuses on logistics and employee performance.

### Includes:

* Shipment Analysis
* Employee Performance
* Delivery Trends
* Freight Analysis
* Regional Orders

---

# Dataset Used

The project uses the Northwind Traders dataset containing:

* Orders
* Order Details
* Products
* Categories
* Customers
* Employees
* Shippers

---

# Key Features

* Interactive slicers and filters
* Multi-page dashboard navigation
* Professional dark theme UI
* KPI cards and trend analysis
* Business insights and storytelling

---

# Key Insights

* Sales show seasonal trends across months.
* Beverages and Dairy Products generate the highest revenue.
* Top products contribute major sales share.
* United Package handled the highest shipment volume.
* Employee performance varies significantly.
